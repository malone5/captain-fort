# captain-fort
AWS lambda function for the "Captain Fort" Alex Skill. 

## Using the skill 

1. Initiate the skill with "**Alexa, ask Captain Fort...**"
2a. Ask for a drop location
   "Give me a drop location?"
   "Where we droppping?"
   "Where should I drop?"
2b. Ask for a training challenge
   "Give me a challenge"
   "Challenge me"
